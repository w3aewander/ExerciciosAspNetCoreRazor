# 💻 Exercícios ASP.NET Core Razor Pages

> Coleção de exercícios práticos desenvolvidos para explorar os recursos do ASP.NET Core utilizando a engine de renderização Razor.

## 📌 Sobre o Projeto

Este repositório serve como um portfólio de aprendizado, contendo a implementação de diversos conceitos fundamentais e avançados do ecossistema .NET voltado para a web. O foco principal é a utilização de **Razor Pages** para a criação de interfaces dinâmicas, manipulação de dados e lógica de servidor.

### 🧪 Conceitos Praticados:

   **Sintaxe Razor:** Manipulação de lógica C# diretamente no HTML.
   **Tag Helpers:** Facilitação da criação de formulários e links.
   **Model Binding:** Vinculação eficiente de dados entre requisições HTTP e modelos C#.
   **Injeção de Dependência:** Configuração e uso de serviços nativos do .NET.
   **Layouts e Partial Views:** Reutilização de componentes de interface.
   **Roteamento:** Configuração de URLs amigáveis e parâmetros de rota.


## 🛠️ Tecnologias Utilizadas

   **Linguagem:** C#

   **Framework:** ASP.NET Core

   **Engine de View:** Razor Pages

   **Estilização:** Bootstrap / CSS3

   **IDE Recomendada:** Visual Studio 2022 ou VS Code



## 📂 Estrutura do Repositório

## 🚀 Como Executar o Projeto

Para rodar este projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/w3aewander/ExerciciosAspNetCoreRazor.git
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd ExerciciosAspNetCoreRazor
    ```

3.  **Restaure as dependências:**
    ```bash
    dotnet restore
    ```

4.  **Execute a aplicação:**
    ```bash
    dotnet run
    ```

5.  **Acesse no navegador:**
    Abra [http://localhost:5000](http://localhost:5000) (ou a porta indicada no terminal).
